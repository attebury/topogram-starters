# Topogram Starter: Hello Web

Package-backed vanilla HTML/CSS/JS starter for `topogram copy`.

```bash
topogram copy @topogram/starter-hello-web ./hello-web
```

## Contract

- Catalog id: `hello-web`
- Surfaces: `web`
- Generator: `@topogram/generator-vanilla-web`
- Runtime stack: vanilla HTML/CSS/JS
- Executable implementation: no
- Purpose: smallest generated web starter with two pages and one workflow.

## First Run

```bash
topogram copy hello-web ./hello-web
cd ./hello-web
npm install
npm run doctor
npm run check
npm run generate
```
